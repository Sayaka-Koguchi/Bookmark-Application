# ① 課題番号-プロダクト名

ブックマークアプリ

## ② 課題内容（どんな作品か）

- index.phpで送信した書籍の内容をinsert.phpにてSQLに接続、挿入、select.phpでSQLのデータを取得、表示させる
- ブックマークにカテゴリー、読書状況の項目を追加、登録するようにした
- 各読書状況ごとに冊数をカウント、表示させるようにした
